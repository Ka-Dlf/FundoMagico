# Fundo Mágico

## Sobre
O Fundo Mágico é um projeto prático e super interessante desenvolvido para explorar a integração do desenvolvimento front-end com inteligência artificial e ferramentas de automação.

A ideia principal é permitir que o usuário descreva o tipo de plano de fundo que imagina numa caixa de texto. Ao clicar no botão, a aplicação faz uma requisição para um webhook de IA (configurado no n8n), que processa a descrição e devolve os códigos HTML e CSS prontos, mostrando uma pré-visualização em tempo real diretamente na página.
